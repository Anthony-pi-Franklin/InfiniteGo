# 游戏UI 更新 - 测试指南

## 测试环境
- 访问游戏主界面（进入一个房间后）

## 功能测试清单

### 1. Color Section 已移除 ✓
- [ ] 左侧菜单中没有 "Color" section
- [ ] 没有颜色选择按钮
- [ ] 菜单显示流畅，没有空白区域

### 2. 小地图（Minimap）嵌入/分离
**初始状态：**
- [ ] 游戏开始时，小地图显示在左侧菜单中（嵌入）
- [ ] 小地图在菜单内，宽度占满菜单宽度

**双击分离：**
- [ ] 双击小地图的标题（"Minimap ▲"）
- [ ] 小地图从菜单中分离出来，浮动在右上角
- [ ] 分离后可以拖拽小地图移动位置
- [ ] 分离后可以通过右下角的句柄调整大小
- [ ] 分离后可以用鼠标在小地图上拖拽导航主画布视图
- [ ] 分离后可以用滚轮缩放小地图

**双击重新嵌入：**
- [ ] 浮动状态下双击标题（"Minimap ▲"）
- [ ] 小地图重新嵌入回菜单中
- [ ] 嵌入后无法拖拽或调整大小

### 3. 领导板（Leaderboard）嵌入/分离
**初始状态：**
- [ ] 游戏开始时，领导板显示在左侧菜单中（嵌入）
- [ ] 领导板在菜单内，宽度占满菜单宽度
- [ ] 领导板根据玩家数量自动调整高度

**双击分离：**
- [ ] 双击领导板的标题（"Leaderboard ▲"）
- [ ] 领导板从菜单中分离出来，浮动在右下角
- [ ] 分离后可以拖拽领导板移动位置
- [ ] 分离后可以通过右下角的句柄调整大小
- [ ] 分离后可以手动调整领导板的高度

**双击重新嵌入：**
- [ ] 浮动状态下双击标题（"Leaderboard ▲"）
- [ ] 领导板重新嵌入回菜单中
- [ ] 嵌入后无法拖拽或调整大小

### 4. 菜单隐藏时组件自动隐藏
**测试菜单隐藏功能：**
- [ ] 点击菜单关闭按钮（✕）或点击菜单外的区域（如果有）
- [ ] 菜单隐藏
- [ ] 嵌入在菜单中的小地图和领导板也隐藏
- [ ] 浮动的小地图和领导板仍然可见

**菜单展开后：**
- [ ] 点击菜单按钮（☰）或其他打开菜单的方式
- [ ] 菜单展开
- [ ] 嵌入的小地图和领导板重新可见

### 5. Leader Board 自动大小调整
**放置石头（增加玩家分数）：**
- [ ] 在游戏中放置多个不同颜色的石头
- [ ] 观察领导板的高度变化
- [ ] 领导板应该根据当前的颜色数量自动调整高度

**高度限制测试：**
- [ ] 如果有超过10个颜色，领导板最多显示前10个
- [ ] 领导板不会无限增长

**手动调整大小后：**
- [ ] 分离小地图/领导板后，手动调整其大小
- [ ] 继续放置石头，增加排名数量
- [ ] 观察：领导板应该保持您设置的大小（不再自动调整）

**重新嵌入后的大小：**
- [ ] 在浮动状态下手动调整大小
- [ ] 双击标题重新嵌入菜单
- [ ] 再次双击分离
- [ ] 应该恢复到默认大小（重置手动调整标志）

## 边界情况测试
